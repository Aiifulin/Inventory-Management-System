Import Products

Import Guidelines

  • Required Columns: Product Name, Category, Price, Stock, Size, Color
  • Optional Columns: Description, Custom Variation, Low Stock Threshold, Attribute Name, Attribute Value
  • Price: Must be a number (e.g., 99.99). Use 0 for free products.
  • Stock: Must be a positive integer (e.g., 50)
  • Category: Must match existing categories exactly (case-sensitive)
  • Duplicates: Products with existing names will be skipped
  • Images: Not supported in import. Add images manually after import.
